🛡️ Privacy-First Features

🔐

End-to-End Encryption

All posts, images, and media are encrypted client-side using ChaCha20-Poly1305. Your private keys never leave your device.

🌐

Peer-to-Peer Network

Direct connections between users via WebRTC. No central server can access your content or metadata.

🖼️

Encrypted Media Sharing

Share images, videos, and files with automatic client-side encryption and decryption.

⚡

Zero-Knowledge Architecture

The server never sees your plaintext data. Everything is encrypted before leaving your browser.

🔑

Public Key Cryptography

Uses NaCl (TweetNaCl) for secure key generation, message encryption, and digital signatures.

🚀

Modern Rails Stack

Built with Rails 8, Hotwire (Turbo + Stimulus), and modern JavaScript for a responsive experience.

⚙️ Technology Stack

Backend

Ruby on Rails 8 SQLite RbNaCl Solid Queue

Frontend

Hotwire (Turbo + Stimulus) TweetNaCl WebRTC Modern CSS

Cryptography

🚀 Quick Setup

1. Clone Repository

git clone https://github.com/your-username/cipher.git
cd cipher

2. Install Dependencies

bundle install

3. Setup Database

bin/rails db:prepare

4. Start Server

bin/rails server

Requirements

Ruby: 3.3+
Rails: 8.0+
Node.js: 18+ (for asset compilation)
Browser: Modern browser with WebRTC support
